# Lancit — 프리랜서 통합 관리 플랫폼

> 프리랜서와 클라이언트를 더 스마트하게 연결합니다

<br>

## 📌 프로젝트 개요

| 항목        | 내용                                           |
| ----------- | ---------------------------------------------- |
| 프로젝트명  | Lancit                                         |
| 개발 기간   | 2026.05.15 ~ 2026.06.25                        |
| 서비스 유형 | 프리랜서 ↔ 클라이언트 매칭 및 업무 관리 플랫폼 |

프리랜서는 다수의 클라이언트로부터 의뢰를 받아 일하는 구조 특성상
계약 관리·일정 관리·수익 추적이 분산되어 있어 비효율이 발생합니다.

Lancit은 **의뢰 수신 → 계약 협상 → 전자서명 → 포트폴리오 축적**까지의
전 과정을 하나의 플랫폼에서 처리할 수 있도록 설계되었습니다.

## 🗂 서비스 구조

서비스는 **프리랜서** / **클라이언트(회사)** 두 가지 회원 유형으로 구분됩니다.

```
Lancit
├── 비로그인
│   ├── 서비스 소개 랜딩 페이지
│   ├── 로그인
│   ├── 회원가입 — 유형 선택 → 프리랜서 / 회사
│   └── 비밀번호 찾기
│
├── 프리랜서
│   ├── 캘린더        월간 일정 관리 · AI 파싱 등록
│   ├── 계약관리      계약 협상 · 계약서 CRUD(전자서명) · PDF 다운로드
│   ├── 제안 관리(회->프)    제안 온 프로젝트 확인 · 수락(수락시 계약관리의 협의 페이지로 이동)/거절
│   ├── 구인페이지(프->회)    구인공고 탐색 · 구인공고 지원하기 (포트폴리오에서 지원할 파일 선택)
│   └── 포트폴리오    등록/수정/삭제 · 파일 첨부 · 공개 비공개 설정
│
└── 클라이언트(회사)
    ├── 캘린더        월간 일정 관리 · AI 파싱 등록
    ├── 계약관리      계약 협상 · 계약서 CRUD(전자서명) ·  PDF 다운로드
    ├── 제안 관리(회->프)    프리랜서 리스트 탐색 · 공개 포트폴리오 열람· 제안서 CRUD · 수락(수락시 계약관리의 협의 페이지로 이동)/거절
    └── 구인페이지(프->회)      구인공고 등록/수정/마감 · 구인공고 지원서 확인하기 · 수락(수락시 계약관리의 협의 페이지로 이동)/거절
```

## ✨ 핵심 기능

### 1. 전자계약 — 갑·을 협상 플로우

단순한 계약서 보관이 아니라 **갑·을이 조건을 협상하는 과정**을 지원합니다.

```
클라이언트(갑)  협의 진행 후 계약서, 전자 서명 작성
      ↓
프리랜서(을)    계약서 수정 후 전자 서명
      ↓
클라이언트(갑)  최종 검토 → 계약 진행
      ↓
계약서 진행 페이지로 이동
      ↓
PDF 다운로드
```

- 프리랜서 계약서 기본 포맷 제공

---

### 2. 클라이언트 사업자 신뢰도 검증

계약 전 클라이언트의 사업자 실재 여부를 **국세청 API**로 자동 확인합니다.

```
사업자등록 번호 데이터 받기
      ↓
국세청 API 조회
      ↓
✅ 정상               → "사업자 확인 완료" 배지
⚠  휴업중 · 폐업 · 사업자 없음  → 주의 경고 표시
```

---

### 3. AI 파싱 — 의뢰 메시지 자동 입력

의뢰 텍스트를 입력하면 주요 필드를 자동으로 추출해 폼을 채워줍니다.

| 파싱 유형   | 입력                               | 추출 필드                        |
| ----------- | ---------------------------------- | -------------------------------- |
| 의뢰 메시지 | 카카오톡 · 이메일 등 텍스트 데이터 | 제목 · 납기일 · 금액 · 의뢰처 등 |

> 파싱 결과는 즉시 저장되지 않으며 사용자 검토 후 확정됩니다.
